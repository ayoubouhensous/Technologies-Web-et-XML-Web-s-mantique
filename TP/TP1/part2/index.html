Project

Application de compression d'images (IRM)

J’ai développé une application de compression d’images spécifiquement conçue pour les images d’IRM (Imagerie par Résonance Magnétique). Cette application utilise des algorithmes avancés de compression pour réduire la taille des fichiers sans compromettre la qualité des images. Grâce à une interface conviviale, les utilisateurs peuvent facilement importer, compresser et exporter les images, optimisant ainsi le stockage et le partage des données médicales.

Application de reconnaissance faciale

J’ai conçu une application de reconnaissance faciale utilisant JavaFX et FaceNet AI. Cette application permet d’identifier et de vérifier l’identité des individus en temps réel. Grâce à des modèles d’apprentissage profond, elle offre une précision élevée et une rapidité d’exécution. L’interface utilisateur est intuitive, permettant aux utilisateurs de capturer des images, d’exécuter des reconnaissances faciales et d’afficher les résultats instantanément, ce qui en fait un outil puissant pour des applications de sécurité et d’accès.

Application web de gestion de laboratoire

J’ai développé une application web de gestion en utilisant Spring Boot pour le backend et Angular pour le frontend. Cette application permet d’optimiser la gestion des patients, des analyses et du personnel médical grâce à une interface intuitive et une architecture robuste.
